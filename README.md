# Overview

CRM is a customer relationship management system to help business owners easily track all clients communications and nurture relationships with their clients.

- *Front End*: HTML, CSS, Bootstrap,JSP
- *Back End*: Java
- *Frameword*: Spring, SpringMVC, Mybatis
- *Database*: MySQL
